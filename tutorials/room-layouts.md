# Room Layouts

Room layouts let you save and restore different furniture arrangements for the same room. This is useful for trying multiple staging options and comparing them.

## Saving a Layout

1. Arrange furniture in the room however you like
2. Tap the **layouts button** (grid icon) in the controls bar
3. Tap **Save Layout** at the bottom of the panel
4. Enter a name and tap **Save**

A screenshot thumbnail is captured automatically so you can visually identify each layout.

## Loading a Layout

1. Open the layouts panel
2. Tap a layout card to load it
3. If the room already has furniture, a confirmation prompt asks before replacing

All saved furniture positions, rotations, and surface orientations are restored. The meter stick is preserved during layout changes.

## Deleting a Layout

Hover over (or long-press on mobile) a layout card to reveal the delete button in the top-right corner. Tap it to remove the layout.

## Notes

- Each room can have up to **10 saved layouts**
- The counter in the panel header shows how many you have
- If a furniture item was deleted from the database since the layout was saved, it will be skipped and you will be notified
- Layouts are saved per room and deleted automatically when the room, house, or org is deleted
- Loading a layout does not auto-save the room — save the room separately if you want to keep the arrangement
