# Exporting a House

Export generates a ZIP file containing screenshots, inventory data, and a staging report for the house.

## How to Export

1. Tap the **House** button in the controls bar
2. Tap **Export House**
3. Wait for the export to process — a progress modal shows each room being captured

## Export Contents

The ZIP file includes:

- **Room screenshots** — one image per room showing the staged scene at full resolution
- **Delivery manifest** — a list of all furniture needed, with per-room breakdown and totals
- **PDF staging report** — a formatted document with room photos and furniture inventory, ready for printing or sharing

## Notes

- Screenshots are captured using an offscreen renderer at the original photo resolution
- The meter stick is not included in screenshots
- All rooms in the house are included in the export
