# House Button

The House button opens the session modal with options for the currently loaded house.

## Options

- **Edit House** — change the house name or date range
- **Export House** — generate a ZIP with room screenshots, delivery manifest, and PDF staging report
- **Close House** — save and return to the calendar
- **Delete House** — permanently delete the house and all its rooms
- **Sign Out** — log out of the current organization account
