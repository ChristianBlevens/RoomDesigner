# Meter Stick

The meter stick is a built-in 1-meter reference object for calibrating room scale. Place it next to something with a known size in the photo, then adjust the room scale slider until the meter stick matches.

## How to Use

1. Tap the **meter stick button** (ruler icon) — it starts pulsing to indicate placement mode
2. Tap a surface in the room to place the meter stick
3. Open the **scale panel** and adjust until the meter stick matches a known-size object
4. Tap the meter stick button again to **hide** it when done

## States

- **Not placed** — button is in default state. Tapping enters placement mode
- **Placement mode** — button pulses blue. Tap a surface to place, or tap the button again to cancel
- **Visible** — meter stick is in the scene. Button highlights. Tapping hides it
- **Hidden** — meter stick exists but is invisible. Tapping shows it again

## Interaction

- The meter stick can be **selected, dragged, and rotated** just like furniture
- **Delete** it via the gizmo delete button to return to the "not placed" state
- It orients automatically — stands upright on floors, lays flat against walls

## Notes

- The meter stick is bright orange to stand out against any room photo
- It is saved with the room and restored when you return
- It does not appear in screenshots or exports
- It is not affected by room scale — it always represents exactly 1 meter
